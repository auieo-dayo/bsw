# 環境変数ドキュメント

BDSのサーバー設定は `.env` ファイルで行います。`sample.env` をコピーして `.env` という名前に変更し、カスタマイズしてください。

---

## 基本設定

### サーバー名 - `server-name`

サーバーの名前を設定します。Minecraft クライアントからサーバー一覧に表示される際の名前です。

| プロパティ   | 型     | デフォルト    | 説明                           |
| -------- | ----- | -------- | ---------------------------- |
| `server-name` | string | BSWserver | サーバーの表示名                  |

**例：**
```
server-name=MyAwesomeServer
```

---

## ゲームモード設定

### ゲームモード - `gamemode`

プレイヤーのデフォルトゲームモードを設定します。

| プロパティ   | 型     | デフォルト | 説明                           |
| -------- | ----- | ------ | ---------------------------- |
| `gamemode` | string | survival | ゲームモード（survival / creative / adventure） |

**使用可能な値：**
- `survival` - サバイバルモード。ブロック採取、飢え、ダメージあり
- `creative` - クリエイティブモード。無制限資材、飛行可能、ダメージなし
- `adventure` - アドベンチャーモード。特定のブロックのみ採取可能

**例：**
```
gamemode=creative
```

---

## 難易度設定

### 難易度 - `difficulty`

サーバーの難易度を設定します。

| プロパティ   | 型     | デフォルト | 説明                           |
| -------- | ----- | ------ | --------------------------------------------------------- |
| `difficulty` | string | normal | 難易度（peaceful / easy / normal / hard） |

**使用可能な値：**
- `peaceful` - 平和。敵モブが出現しない
- `easy` - イージー。敵モブが出現し、ダメージが軽い
- `normal` - ノーマル。敵モブが出現し、ダメージが通常
- `hard` - ハード。敵モブが出現し、ダメージが大きい

**例：**
```
difficulty=hard
```

---

## チート設定

### チート許可 - `allow-cheats`

プレイヤーがコマンドを実行できるかを設定します。

| プロパティ    | 型      | デフォルト | 説明                           |
| --------- | ------ | ------ | ---------------------------- |
| `allow-cheats` | boolean | true   | チート（コマンド）を許可するか |

**使用可能な値：**
- `true` - コマンドを許可
- `false` - コマンドを禁止

**例：**
```
allow-cheats=false
```

---

## プレイヤー設定

### 最大プレイヤー数 - `max-players`

サーバーに同時接続できるプレイヤーの最大数を設定します。

| プロパティ    | 型     | デフォルト | 説明                           |
| --------- | ------ | ------ | ---------------------------- |
| `max-players` | number | 20     | 最大接続プレイヤー数             |

**例：**
```
max-players=50
```

---

## ネットワーク設定

### サーバーポート（IPv4） - `server-port`

サーバーが使用する UDP ポート番号（IPv4）です。

| プロパティ     | 型     | デフォルト | 説明                           |
| --------- | ------ | ------ | ---------------------------- |
| `server-port` | number | 19132  | サーバーポート（IPv4）            |

**例：**
```
server-port=19132
```

### サーバーポート（IPv6） - `server-portv6`

サーバーが使用する UDP ポート番号（IPv6）です。

| プロパティ      | 型     | デフォルト | 説明                           |
| ---------- | ------ | ------ | ---------------------------- |
| `server-portv6` | number | 19133  | サーバーポート（IPv6）            |

**例：**
```
server-portv6=19133
```

---

## ワールド設定

### ワールド名 - `level-name`

ワールドの名前を設定します。サーバーデータディレクトリ内のフォルダ名として使用されます。

| プロパティ     | 型     | デフォルト | 説明                           |
| --------- | ------ | ------ | ---------------------------- |
| `level-name` | string | world  | ワールド名                      |

**例：**
```
level-name=SkyBlock
```

### ワールドシード - `level-seed`

ワールド生成のシードを設定します。同じシードでワールドを作成すると、同じ地形が生成されます。空の場合はランダムに生成されます。

| プロパティ      | 型     | デフォルト | 説明                           |
| ---------- | ------ | ------ | --------------------------------------------------------- |
| `level-seed` | string | 空     | ワールドシード（数値または文字列。空でランダム生成） |

**例：**
```
level-seed=12345
```

または

```
level-seed=MyCustomSeed
```

---

## セットアップ手順

1. `sample.env` をコピーしてファイル名を `.env` に変更します
2. `.env` をテキストエディタで開き、必要な設定を変更します
3. 各プロパティは `key=value` の形式で設定します
4. コメント行は `#` で始まります
5. 設定が完了したら、BSWを起動すると新しい設定が適用されます

---

## 設定例

以下は一般的な設定例です：

```
# サーバー基本情報
server-name=MyMinecraftServer
gamemode=survival
difficulty=normal
allow-cheats=true
max-players=20

# ネットワーク設定
server-port=19132
server-portv6=19133

# ワールド設定
level-name=world
level-seed=
```

---

## 注意事項

- ポート番号は 1-65535 の範囲で設定してください
- `server-port` と `server-portv6` は異なるポート番号にしてください
- ファイアウォール設定によっては、ポートが外部からアクセスできないことがあります
- シードを空にするとランダムにワールドが生成されます
